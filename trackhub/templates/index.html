Access Model

The public home page stays informative. Every location page and subproject path goes through location selection first unless that location has already been authenticated in the current session.

  • Home page stays public and descriptive.
  • Choose a location before entering any subproject.
  • Each location requires its own password.
  • The shell then routes into stable paths like /map3d/.

Routing

The preferred production shape is one hostname and stable subpaths. The shell currently supports app-proxy for convenience and can later sit in front of a real reverse proxy.
