## Naming guidance

- Keep names broad enough to cover the whole project, not just one subproject
- Favor names that fit:
  - archival capture
  - technical documentation
  - control / operations
  - modular growth
- Avoid locking the name too tightly to:
  - only networking
  - only 3D mapping
  - only museum use
