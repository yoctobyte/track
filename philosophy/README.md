# Philosophy

This folder holds project philosophy, intent, and directional notes.

Use this folder for:

- project vision
- design philosophy
- conceptual framing
- notes extracted from broader planning conversations

Do not use this folder for:

- implementation details
- route/API inventories
- current technical status
- task logs

Those belong in files such as `GOALS.md`, `PROGRESS.md`, or subproject-local
technical notes.
